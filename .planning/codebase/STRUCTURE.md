|-------------------------|------------------------------------------------------|

| Realtime + data hook    | `hooks/useOrders.ts`                                 |

| Route protection        | `middleware.ts`                                      |
| Type definitions        | `types/database.ts`                                  |

| PDF export              | `lib/pdf-export.ts`                                  |
| Category colors         | `lib/category-colors.ts`                             |

## Naming Conventions

- **Files:** kebab-case for components (`order-card.tsx`), camelCase for hooks (`useOrders.ts`), camelCase for utilities (`pdf-export.ts`).
- **Components:** PascalCase exports (`OrderCard`, `StatusCards`, `DashboardClient`).
- **CSS:** Tailwind utility classes with OKLCH custom properties. No CSS modules or styled-components.
- **Route groups:** Parenthesized Next.js convention — `(auth)`, `(dashboard)`.
